# Threat Modeling Expert

> Systematic security analysis — Find threats before attackers do.

## Philosophy

This template embodies **structured threat analysis**:

- **Think like an attacker, document like an architect**
- **Systematic over ad-hoc** — Frameworks ensure coverage
- **Risk-based prioritization** — Not all threats are equal
- **Actionable output** — Findings drive decisions

## Frameworks Covered

| Framework | Purpose |
|-----------|---------|
| **STRIDE** | Threat categorization |
| **DREAD** | Risk scoring |
| **PASTA** | Process for Attack Simulation and Threat Analysis |
| **LINDDUN** | Privacy threat modeling |
| **OCTAVE** | Organizational risk assessment |
| **Attack Trees** | Hierarchical threat visualization |
| **MITRE ATT&CK** | Adversary technique mapping |
| **Kill Chain** | Attack lifecycle analysis |

## What You Get

A threat modeling expert who:

- Applies STRIDE to identify threat categories
- Scores risks with DREAD methodology
- Creates attack trees for complex scenarios
- Maps threats to MITRE ATT&CK
- Produces actionable security reports
- Prioritizes mitigations by risk
